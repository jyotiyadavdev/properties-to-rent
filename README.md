# Rentals on Market

A modern real estate rentals web application built with **Vite** and **React**, using a rich UI component stack (MUI, Radix UI, Tailwind CSS utilities, and more).

## Tech Stack

- **Build tool**: Vite
- **Framework**: React (18.x)
- **UI libraries**: MUI, Radix UI, Lucide icons
- **Styling utilities**: Tailwind CSS, tailwind-merge
- **State & forms**: React Hook Form and related utilities

## Getting Started

### Prerequisites

- **Node.js** 18 or newer installed on your machine
- **npm** (bundled with Node.js)

### Installation

1. Open a terminal in the project folder:
   ```bash
   cd "Real Estate Website Development (3)"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Run the Development Server

Start the Vite dev server:

```bash
npm run dev
```

By default, the app will be available at:

```text
http://localhost:5173/
```

Open this URL in your browser to view the site.

### Build for Production

To create an optimized production build:

```bash
npm run build
```

The compiled assets will be generated in the `dist` folder.

## Project Structure (High Level)

- `src/` – React components, pages, and main application code
- `index.html` – Root HTML file used by Vite
- `vite.config.ts` – Vite configuration
- `utils/` – Utility helpers and shared logic
- `supabase/` – Supabase-related configuration or scripts (if used)
